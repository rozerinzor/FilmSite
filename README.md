# 🎬 Film Website

Bu proje **HTML ve CSS kullanılarak geliştirilmiş responsive bir film web sitesidir.**
Modern ve sade bir tasarımla hazırlanmış olup kullanıcıların film içeriklerini kolayca keşfetmesini sağlayan bir arayüz sunar.

## 🚀 Proje Hakkında

Film Website projesi, film içeriklerini sergilemek için tasarlanmış modern bir web arayüzüdür.
Site **responsive tasarım** kullanılarak geliştirilmiştir ve mobil, tablet ve masaüstü cihazlarda uyumlu şekilde çalışır.

## 💻 Projeyi Çalıştırma

Projeyi bilgisayarında çalıştırmak için:

1. Repoyu klonla

```
git clone https://github.com/rozerinzor/FilmSite.git
```

2. Proje klasörüne gir

```
cd FilmSite
```

3. `index.html` dosyasını tarayıcıda aç.
